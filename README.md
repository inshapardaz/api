# inshapardaz
Language Services for Urdu

# Components

1. Rest API
2. Database
3. Front-End

## Tools and Frameworks requires

### API
- Dotnet Core (https://www.microsoft.com/net/core)
- SQL Server 2016
- Node JS for client & UI develoment

## Building

### Front End

`npm install`

`grunt`

### Rest Service and WebSite
Checkout code and 

`dotnet restore`

`dotnet build`

`dotnet run`
